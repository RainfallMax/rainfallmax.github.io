### 一、安装
官网选择合适方式安装
[Git](https://git-scm.com/)

### 二、 创建仓库
* 创建并选中仓库文件夹
```
$ mkdir learn-git
$ cd learn-git
```

通过git init命令把这个目录变成Git可以管理的仓库
```
git init
```

### 三、添加文件(add)
第一步，使用命令**git add <file>**，注意，可反复多次使用，添加多个文件；
```
git add readme.txt
```
第二步，使用命令**git commit**，完成。
```
git commit -m "wrote a readme file"
```
git commit命令，-m后面输入的是本次提交的说明，能从历史记录里方便地找到改动记录。
### 四、查看版本历史(log)
```
git log

git log --oneline
```

### 五、回溯版本(reset)
有两个提交信息，现在需要回退版本
> 586bead (HEAD -> master) use git  
> d2142d7 use git  

回退到指定版本
```
git reset --HEAD d2142d7
```

查看所有记录
```
git reflog

输出示例
586bead (HEAD -> master) HEAD@{0}: commit: use git
d2142d7 HEAD@{1}: commit (initial): use git
```

回到现在版本
```
git reset --hard d2142d7
或者
git reset --hard HEAD@{1}
```

### 六、单个文件回溯（checkout）

```
git checkout d2142d7 --file
```
 Git  checkout 回退版本+文件


### 七、分支
创建dev分支
```
git branch dev
```
查看所有分支
```
git branch
```
切换分支
```
git checkout dev
```
删除分支
```
git branch -d dev
```
