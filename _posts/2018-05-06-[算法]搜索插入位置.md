# Search Insert Position 搜索插入位置

Given a sorted array and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order.

You may assume no duplicates in the array.

```
输入: [1,3,5,6], 5
输出: 2

输入: [1,3,5,6], 2
输出: 1

输入: [1,3,5,6], 7
输出: 4

输入: [1,3,5,6], 0
输出: 0
```

## python3 
我的思路
顺序列表用二分查找，如下。时间 O(log n)
```
class Solution:
    def searchInsert(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        low =0
        high =len(nums)-1
        while low<=high:
            mid=(low+high)//2
            guess=nums[mid]
            if guess == target:
                return mid
            if guess>target:
                high=mid-1
            if guess<target:
                low=mid+1
        return low
```

### discuss里其他解法
1.>>>返回数组中小于target部分的长度。时间O(n)
```
class Solution(object):
    def searchInsert(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """       
        return len([x for x in nums if x<target])
```

2.>>>python 自带二分查找（我草泥马！）
python标准库中有一个模块，bisect。这个库接受有序的序列，内部实现就是二分。

```
class Solution(object):
    def searchInsert(self, nums, target):
        """
        :type nums: List[int]
        :type target: int
        :rtype: int
        """
        return bisect.bisect_left(nums, target)
```

```
import bisect  
bisect.bisect_left(nums,x) 在nums列表中查找x，若存在，返回x左侧位置  
bisect.bisect_right(nums,x)  
bisect.insort_left(nums,x) 在nums列表中查找x，若存在，插入x左侧；  
bisect.insort_right(nums,x)
```
